Master Design Project

Building 3D maps using a manually driven robot for use in helping to guide various autonomous robots.
Back to resume and portfolio

Project Details

During this project, our team aimed to modify a robot in order to create dense high resolution 3D maps of buildings at a reasonable price. With this map, the client would be able to overlay other information about a building that we gather, such as Wifi strength and temperature, to create a intuitive interface for buliding managers. This interface would also be able to communicate with sensors in the building giving live readings for metrics such as water pressure, and would track the location of the building's fleet of autonomous robots. Ideally, this 3D map would be able to be sliced into relevant sections to improve the localization for the other autonomous robots in the building, and would be useful regardless of whether they are the size of a Roomba or the size of an Intellibot.

Early Map

This is an example of one of the earlier maps we made during our initial testing just using RGB-D without odometry. This is a view of a hallway the robot drove through. The camera was mounted onto a Clearpath Jackal robot using a custom mount, and the robot was driven around using a PS3 controller. The initial tests look promising, but we aim to improve the resolution of the map by integrating other sensors.
